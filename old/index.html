programmingmath by jostylr

Learning math via programming javascript

This is for learning and exploring math using the power of JavaScript. The intention of this project is to put all the functions and code in a nice accessible way. Kind of a library with documentation, but its main purpose is as a staging area before deployment to the blog programmingmath.com

  • Shuffling
  • Factorial
  • Exact Arithmetic
